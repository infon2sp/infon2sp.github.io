Tentang Open Project

Open project adalah program kolaborasi terbuka dari Media Info N2SP untuk fotografer, model, dan kreator yang ingin terlibat dalam project fotografi bersama. Kegiatan ini bertujuan untuk meningkatkan portofolio dan memperluas jaringan.

Project Aktif

Street Photography Braga

Hunting foto street di kawasan Braga dengan tema urban & candid moment.

Status: Open

Portrait Session Collaboration

Kolaborasi fotografer dan model untuk sesi portrait kreatif.

Status: Open

Event Dokumentasi Komunitas

Project dokumentasi kegiatan komunitas dan event lokal.

Status: Open

Cara Ikut Project

  • Pilih project yang tersedia
  • Hubungi tim melalui kontak
  • Konfirmasi keikutsertaan
  • Ikut kegiatan sesuai jadwal

Manfaat Mengikuti Project

  • Menambah portofolio fotografi
  • Relasi dengan fotografer & kreator lain
  • Pengalaman project nyata
  • Kesempatan kolaborasi lanjutan
